## [Unreleased]

### Added
- Core `Tracer` class with `@tracer.trace` decorator and `tracer.span()` context manager
- `ContextVar`-based async-safe context propagation (works correctly with `asyncio.gather`)
- `SpanContext` with `set_input()`, `set_output()`, `set_metadata()`, `record_error()`
- SQLite-backed `TraceStore` with WAL mode at `~/.traceai/traces.db`
- Pydantic v2 models: `Span`, `Trace`, `SpanKind`, `SpanStatus`, `ErrorDetail`
- Framework-agnostic tracing — works with any Python code, no framework required
- `traceai.instrument()` dispatcher for future auto-instrumentation
- 66 tests across models, storage, and tracer (all passing)
- Example scripts: `basic_trace.py`, `custom_agent.py`
- GitHub repository maintenance: CI, release workflow, issue templates, PR template
